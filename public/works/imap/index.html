Интерактивная карта

Описание:

Интерактивная карта, с нанесенными на нее маркерами городов, в которых находится интересующий объект. Маркеры ставятся по мировым координатам. Карта нарисована с помощью topojson.js и d3.js. Есть возможность задавать данные для областей и регионов, а именно:
  • - цвет фигуры;
  • - название области или региона;
  • - города, которые нужно отметить маркером;
Чтобы установить метку, необходимо для конкретного регина в поле "cities" вписать данные города, к примеру: 
  {
      "cityName": "Москва", // Название города
      "lat": "55.7522200", // Latitude
      "lon": "37.6155600", // Longitude
      "link" : "http://www.google.com" // Ссылка куда-либо
  }
